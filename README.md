# RTX-Shader-for-Psych-Engine
Not sure why someone would need this but I got bored and made it so do whatever you want.

Save this file in your mods/shaders/ folder as RTX.frag.
Because this shader relies on active parameters to change values on the fly, you need a Lua script to load it, so place this script inside mods/data/your-song/RTX.lua.
